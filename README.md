# 🌾 农场主小游戏

> 一个基于 React + Phaser 的模拟经营农场游戏

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000

## 🛠️ 技术栈

- **前端**: React 18 + TypeScript + Vite
- **游戏引擎**: Phaser 3
- **状态管理**: Zustand
- **UI组件**: Ant Design
- **样式**: TailwindCSS
- **后端**: Supabase
